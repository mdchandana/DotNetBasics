


The System.Text.Json namespace provides functionality for 
serializing to and deserializing from JavaScript Object Notation (JSON). 
The System.Text.Json library is included in the runtime for .NET Core 3.1 and later versions.
For other target frameworks, install the System.Text.Json NuGet package. The package supports:

.NET Standard 2.0 and later versions
.NET Framework 4.7.2 and later versions
.NET Core 2.0, 2.1, and 2.2
